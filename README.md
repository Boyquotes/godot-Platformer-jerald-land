# JERALD_LAND

A platformer made in Godot

Developed with [Godot-4 Stable](https://godotengine.org/ "Download")

# Features

## blocks

* default block
* spikes
* checkpoints
* jump pads
  
## level editor

* place tiles
* destroy tiles
* move player
* import/export levelst to clipboard
* playtest
* edit block property
* pick block
* pick block with properties
